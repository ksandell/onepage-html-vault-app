# Onepage HTML Vault Creator App
Creates "one page" self-contained HTML pages that contain encrypted files in a "vault".

 files on USB sticks or other unsecure media, while retaining the security around the files.

The vaults are Perfect andextreamly useful for distributing files securely, like encrypted 
zip files, but can be opened by anyone with a modern browser.

**Features**

* Utilizes browser built in crypto libaries
* 2 separate codes needed (multi-key)
